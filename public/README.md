# Gofusa Studio - Static HTML Website

This directory contains a complete static HTML, CSS, and JavaScript version of the Gofusa Studio interior design website.

## Files Structure

```
static-html/
├── index.html              # Home page with all sections
├── privacy-policy.html     # Privacy Policy page
├── terms-of-service.html   # Terms of Service page
├── cookie-policy.html      # Cookie Policy page
├── advertising-policy.html # Advertising Policy page
├── disclaimer.html         # Disclaimer page
├── css/
│   └── styles.css         # Complete stylesheet
├── js/
│   └── main.js            # JavaScript for interactivity
└── README.md              # This file

## Features

### Home Page (index.html)
- Responsive header with scroll effects
- Hero section with full-screen background
- About section with company stats
- Services section with 6 service cards
- Portfolio section with image lightbox
- Testimonials section with client reviews
- Contact form with validation
- Footer with social links

### Legal Pages
All legal pages include:
- Sticky sidebar navigation
- Table of contents with scroll tracking
- Collapsible sections for easy reading
- Mobile-responsive layout
- Back to home link
- Contact information section

### Interactive Features (main.js)
- Smooth scrolling navigation
- Mobile menu toggle
- Portfolio lightbox with keyboard navigation
- Collapsible sections on legal pages
- Active section highlighting
- Contact form submission handling
- Responsive header on scroll

### Styling (styles.css)
- Modern gradient backgrounds
- Professional color scheme (Amber & Gray)
- Responsive grid layouts
- Smooth transitions and animations
- Mobile-first design
- Consistent spacing system

## Browser Compatibility

The website is compatible with all modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Usage

Simply open `index.html` in any web browser to view the website. All pages are linked and fully functional without requiring a server.

For production deployment:
1. Upload all files to your web server
2. Ensure directory structure is maintained
3. Update contact information and links as needed
4. Test all pages and forms

## Customization

- **Colors**: Edit CSS variables in `styles.css` (lines 6-25)
- **Content**: Update HTML files directly
- **Images**: Replace Pexels URLs with your own images
- **Contact Info**: Update email and phone numbers in all HTML files

## Notes

- Images are currently hosted on Pexels CDN
- Contact form requires backend integration for actual email sending
- Social media links are placeholder URLs
- All content is for demonstration purposes
